# Algoritmo No Determinista 

## Estructura del Proyecto

El proyecto está organizado en los siguientes archivos:

- `Main.java`: Punto de entrada del programa.
- `Grafica.java`: Clase para manejar gráficas (vértices y aristas).
- `Vertice.java`: Clase para representar un vértice.
- `Arista.java`: Clase para representar una arista con peso.
- `LectorDatos.java`: Clase para leer datos desde archivos de texto.
- `TrayectoriaNoDeterminista.java`: Implementación del algoritmo para el problema de Trayectoria.
- `ParticionNoDeterminista.java`: Implementación del algoritmo para el problema de Partición.


## Compilación

1. Navega a la carpeta donde están los archivos `.java`.
Geronimo Leslie/ SRC / 
2. Compila todos los archivos con el siguiente comando:

   javac *.java

-- PARA LA TRATECTORIA: 
java Main trayectoria <archivoEntrada> <inicio> <fin> <pesoMinimo>
- El archivo de entrada es grafica.txt 
Ejemplo: java Main trayectoria grafica.txt A D 15


-- PARA LA PARTICION
java Main particion <archivoEntrada>
- El archivo de entrada es "conjunto.txt"
