ESPECIFICACIONES DEL PROGRAMA:
Para ejectutarlo, primero hay que compilar ambos archivos (AlgortimosDeOrdenamiento.java y PruebasOrdenamiento.java), lo puedes hacer 
con javac AlgortimosDeOrdenamiento.java  PruebasOrdenamiento.java o si lo prefieres y no hay má archivos que compilar con: javac *.java.

Una vez  Compilados, ejectuta el programa que contiene el MAIN, que en este caso es: java PruebasOrdenamiento 

Aparecerán los ordenamientos del arreglo original que está especificado en el MAIN (tanto el arreglo INT como el de 
hexadecimal), lo puedes cambiar si así lo prefieres. :D
